## Key Design Patterns

### Hero Sections
- Large uppercase headings (Montserrat/Jost 700-800 weight)
- Subheading with description
- CTA button (coral/red accent)
- Background image with dark overlay

### Overlap Cards
- 3 cards with icons
- Red/coral background color
- White text and icons
- Positioned to overlap hero section

### Service Cards
- Image at top
- Title in bold
- Brief description
- "Read More" link

### Team Member Cards
- Square profile photo
- Name (bold)
- Role title
- Social media icons (Facebook, X/Twitter, LinkedIn)

### Statistics Counters
- Large animated numbers
- Label text below
- Often on accent color background

### Testimonials
- Client photo (circular)
- Quote text
- Name and title
- Carousel navigation dots

### Pricing Tables
- Plan name header
- Feature bullet list
- Price display
- CTA button

### FAQ Section
- Accordion/toggle format
- Question as header
- Answer revealed on click

## Color Palette
- **Primary Navy:** #0E2B48
- **Accent Coral/Red:** #FF1654
- **Gold Highlight:** #d4a54c
- **White:** #FFFFFF
- **Dark Text:** #1a1a2e

## Typography
- **Headings:** Montserrat or Jost, 700-800 weight, UPPERCASE
- **Body:** Open Sans or Roboto, 400 weight, 1.8 line-height

## Key CSS Classes (from our implementation)
- `.exodus-hero` - Main hero section
- `.exodus-about` - About section styling
- `.exodus-services` - Services grid
- `.exodus-languages` - Languages section
- `.overlap-cards-section` - Overlap cards container
- `.overlap-card` - Individual overlap card
- `.exodus-header` - Navigation header
- `.team-section` - Team member section
- `.team-member` - Individual team card
